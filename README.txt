I decided to apply what I have learned from my latest project using
Node js, express and mongoose as my wrapper for the mongodb database.

start the server through cmd: ...\back-end-challenge>node server.js
I used postman to send http requests (interact with API) and 
mongodbCompass for quick analysis.

mongodb url: mongodb://localhost:27017 (configuration in db.config.js)
web server port: http://localhost:8080    (configuration in server.js)
